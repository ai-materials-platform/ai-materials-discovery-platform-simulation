# AI 합금 디지털 트윈 시뮬레이션

조성 비율로 예측한 물성값을 기반으로 합금 모델을 생성하고, 가상 3D 공간에서 물성 테스트를 실행하는 Electron 데스크톱 애플리케이션입니다.

---

## 실행

```bash
npm install
npm start    # 프로덕션 빌드 후 Electron 앱 실행
npm run dev      # 개발 서버 + Electron 동시 실행
```

> `index.html`을 브라우저에서 직접 열면 동작하지 않습니다. Vite/React 번들과 Electron IPC가 필요합니다.

---

## 프로젝트 구조

```
├── electron/        Electron 메인 프로세스 (창 생성, Python 백엔드 자동 실행)
├── src/
│   ├── App.jsx      메인 UI + React Three Fiber 3D 뷰포트
│   └── styles.css   다크 테마 레이아웃
├── backend/         Python 물성 예측·시뮬레이션 API (표준 라이브러리)
└── dist/            npm run build 결과물
```

## 합금 관리

- **프리셋 3종**: Ni-Ti-Cr (고강도), Al-Mg-Si (경량), Fe-Cr-Ni (내열)
- 조성 비율 슬라이더 (원소별 %)
- 밀도 계수, 모델 스케일 조절
- 공정 조건: 용체화 온도 / 처리 시간 / 테스트 온도
- 상태 저장 / 불러오기

---

## AI 물성 예측

백엔드 API 또는 사전학습 RF 모델(`ai-materials-discovery-platform`)로 다음 값을 예측합니다.

| 출력 | 설명 |
|------|------|
| 밀도 | g/cm³ |
| 인장강도 (UTS) | MPa |
| 탄성 계수 | GPa |
| 열전도율 | W/m·K |
| 용융점 | °C |
| 예측 신뢰도 | % |
| 격자 안정성 | % |

예측 결과는 시뮬레이션 입력(파단 임계값, 변형 스케일, 응력-변형률 곡선)으로 직접 사용됩니다.

## 3D 시뮬레이션 뷰포트

### 도형 선택

| 도형 | 설명 |
|------|------|
| 구 | 등방성 합금 기본 모델 |
| 정육면체 | 블록형 구조 |
| 직육면체 | 판재/봉재 형태 |
| 시편 (ASTM) | LatheGeometry 기반 도그본 시편, 다중 그랩 포인트 |
| 격자 | BCC 결정 구조 시각화 |

### 시각화 모드

- **열 분포**: 주황 계열 방출광
- **응력 분포**: 빨간 계열 강조
- **X-Ray**: 반투명 와이어 표면
- **와이어프레임**: 메시 구조 확인

### 물성 테스트

| 테스트 | 작동 방식 |
|--------|-----------|
| 강도 | UTS 기반 최대 응력, 파손 위험 평가 |
| 휘어짐 | 탄성 계수 기반 굽힘 변형 |
| 늘어짐 | 연신율 기반 인장 변형 |
| 온도 | 용융점·열전도율 기반 열 분포 |

테스트 종류에 따라 3D 모델의 스케일(testScale)이 자동 조정되어 카메라 모드와 변형 모드 간 크기가 동일하게 유지됩니다.

---

## 변형 테스트 모드

우측 상단 **변형 테스트** 버튼으로 진입합니다.

### 다중 그랩 변형

- 물체 위 아무 지점을 클릭·드래그하면 그 지점 기준으로 Gaussian 가중치 변형이 적용됩니다.
- 여러 지점을 순서대로 잡아 복합 변형이 가능합니다.
- 드래그 방향은 **카메라 기준**으로 계산되어 (right/up 벡터 투영) 어느 각도에서 봐도 직관적으로 동작합니다.
- 반대편은 Gaussian 감쇠(`σ`)로 자동 고정됩니다 — 별도 핀 불필요.

### 파단 및 찢기

- 최대 변위가 임계값(`DEFORM_FRACTURE`)을 초과하면 파단이 발생합니다.
- **클리핑 플레인 방식 사용 안 함**: 버텍스 레벨에서 상/하 조각으로 분리합니다.
- 파단 경계면 근처 버텍스에 랜덤 노이즈가 적용되어 자연스러운 불규칙 찢김 표면이 생성됩니다.
- 찢어진 위치(그랩 포인트)에 따라 파단 형태가 달라집니다.
- 파단 후 **애니메이션 없음**: 두 조각이 즉시 분리된 상태로 고정됩니다.
- 분리된 두 조각 각각을 독립적으로 드래그할 수 있습니다.

### 상태 유지

- **카메라 이동** 버튼을 눌러도 변형 상태가 초기화되지 않습니다.
- **초기화** 버튼으로 명시적으로 리셋할 수 있습니다.

## 분석 패널

### 응력-변형률 그래프

합금 물성 예측값을 바탕으로 실시간 계산된 곡선을 표시합니다.

- **탄성 구간**: `σ = E × ε` 선형 상승
- **항복 구간**: 소성 변형 시작, 항복점 수직 마커 (주황)
- **변형경화**: UTS까지 완만한 상승 (`σ^0.52` 지수 형태)
- **네킹/파단**: UTS 이후 응력 감소, UTS 마커 (빨강)

그래프 하단에 항복강도(MPa) · UTS(MPa) · 파단 연신율(%) 실수치를 표시합니다.

### 온도 분석 그래프

시뮬레이션 온도 이력 차트.

### 시뮬레이션 통계

| 항목 | 내용 |
|------|------|
| 최대 응력 | MPa |
| 파손 위험 | 평가 텍스트 |
| 예측 신뢰도 | % |
| 최대 변형률 | % (변형 테스트 시 실시간) |
| 그랩 포인트 | 개수 |
| 재료 상태 | 탄성 / 소성 / 파단 |

### 실시간 로그

조작 이력·예측 결과·시뮬레이션 상태를 순서대로 기록합니다.

## 백엔드 API

기본 주소: `http://127.0.0.1:8765`

| 엔드포인트 | 역할 |
|------------|------|
| `GET /health` | 헬스체크 |
| `GET /platform/status` | 외부 모델 연결 상태 |
| `POST /predict` | 로컬 물성 예측 |
| `POST /platform/predict` | 사전학습 RF 모델 예측 |
| `POST /simulate` | 시뮬레이션 결과 계산 |
| `POST /import-prediction` | 외부 예측 결과 임포트 |

요청 예시:

```json
{
  "composition": { "Ni": 42, "Ti": 31, "Cr": 27 },
  "densityScale": 0.62,
  "testType": "strength",
  "scale": 1.0
}
```

## 외부 모델 연결

백엔드는 아래 경로의 사전학습 모델을 자동 탐색합니다.

```
c:\ai_project\ai-materials-discovery-platform-simulation\_external\ai-materials-discovery-platform\models
```

다른 경로를 사용하려면 환경변수로 지정합니다.

```powershell
$env:AI_MATERIALS_PLATFORM_DIR="C:\path\to\ai-materials-discovery-platform"
npm start
```
