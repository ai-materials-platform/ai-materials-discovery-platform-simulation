# 합금 디지털 트윈 시뮬레이션

조성 비율 기반 물성 예측과 3D 가상 시편 시뮬레이션을 통합한 Electron 데스크톱 애플리케이션입니다.  
합금 성분을 설정하면 물성값을 즉시 예측하고, ASTM 도그본 시편 또는 다양한 3D 형상에 변형·파단 시뮬레이션을 직접 수행할 수 있습니다.

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| 조성 비율 입력 | 원소별 슬라이더로 합금 성분(%) 실시간 조정 |
| 물성 예측 | 밀도·인장강도·탄성계수·열전도율·용융점 등 자동 계산 |
| 3D 시편 시뮬레이션 | ASTM E8 도그본 시편에 인장·굽힘·강도 시험 적용 |
| 컵-앤드-콘 파단 | 오스테나이트계 스테인리스강 실제 파단과 유사한 3D 파단면 생성 |
| 응력–변형률 그래프 | 탄성→항복→변형경화→네킹 곡선 실시간 표시 |
| 하단 타임라인 | 변형 이력 재생 및 이벤트 마커 확인 |

---

## 실행

```bash
npm install
npm start        # 프로덕션 빌드 후 Electron 앱 실행
npm run dev      # Vite 개발 서버 + Electron 동시 실행 (핫 리로드)
```

> `index.html`을 브라우저에서 직접 열면 동작하지 않습니다. Vite 번들과 Electron IPC가 필요합니다.


## 합금 설정

- **프리셋 3종**: Ni-Ti-Cr (고강도), Al-Mg-Si (경량), Fe-Cr-Ni (내열)
- 원소별 조성 슬라이더 (%, 합계 자동 정규화)
- 밀도 계수 및 모델 스케일 조절
- 공정 조건: 용체화 온도 / 처리 시간 / 시험 온도
- 설정 내보내기 / 불러오기

## 물성 예측

백엔드 Python API 또는 외부 사전학습 모델(`ai-materials-discovery-platform`)로 아래 물성을 예측합니다.

| 출력값 | 단위 |
|--------|------|
| 밀도 | g/cm³ |
| 인장강도 (UTS) | MPa |
| 탄성계수 | GPa |
| 열전도율 | W/m·K |
| 용융점 | °C |
| 예측 신뢰도 | % |
| 격자 안정성 | % |

예측 결과는 시뮬레이션의 파단 임계값·변형 스케일·응력–변형률 곡선에 직접 반영됩니다.

## 3D 시뮬레이션

### 도형

| 도형 | 설명 |
|------|------|
| 시편 (기본) | LatheGeometry 기반 ASTM E8 도그본, 컵-앤드-콘 파단 지원 |
| 구 | 등방성 합금 기본 모델 |
| 정육면체 | 블록형 구조 |
| 직육면체 | 판재·봉재 형태 |
| 격자 | BCC 결정 구조 시각화 |

### 시각화 모드

- **열 분포**: 주황 계열 방출광
- **응력 분포**: 빨간 계열 강조
- **X-Ray**: 반투명 이중면 렌더링
- **와이어프레임**: 메시 구조 확인

### 물성 테스트

| 테스트 | 물리 모델 |
|--------|-----------|
| 강도 | Y축 인장, Poisson 수축, 게이지 구간 집중 네킹 |
| 굽힘 | 3점 굽힘 단순지지 빔, X 방향 처짐, 섬유 신장 |
| 늘어짐 | Y축 인장 + Gaussian 자유 변형 |
| 온도 | 용융점·열전도율 기반 온도 분포 시각화 |

## 변형 및 파단

### 시편 변형 흐름

1. **탄성 구간** — 낮은 하중: 형상 회복 가능
2. **소성 변형** — 게이지 구간 연신, Poisson 수축으로 단면 감소
3. **네킹** — 약 55 % 하중 이후 게이지 중앙부 급격 수축
4. **파단** — 임계 변위 도달 시 자동 파단

### 컵-앤드-콘 파단 (시편 전용)

실제 오스테나이트계 스테인리스강 인장 시험과 유사한 파단면을 생성합니다.

- **상부 조각 (Cup)**: 외측 림이 아래로 휘어 컵 형태의 오목한 전단 립(shear lip) 생성
- **하부 조각 (Cone)**: 중심축이 위로 솟아 원뿔 형태의 볼록면 생성
- 양쪽 파단면에 결정립 결함과 표면 조도를 모사하는 결정론적 노이즈 적용
- 파단 후 두 조각을 각각 독립적으로 드래그 가능

### 일반 도형 파단

구·정육면체·직육면체 등은 `buildTornGeometry`(평면 절단 + 노이즈 tear surface)로 파단됩니다.

## 분석 패널

### 응력–변형률 그래프

| 구간 | 모델 |
|------|------|
| 탄성 | σ = E × ε (선형) |
| 항복 | 항복점 수직 마커 (주황) |
| 변형경화 | σ ∝ ε^0.52 (멱함수) |
| 네킹/파단 | UTS 이후 응력 감소, UTS 마커 (빨강) |

### 시뮬레이션 통계

| 항목 | 설명 |
|------|------|
| 최대 응력 | MPa |
| 파손 위험 | 정성적 평가 |
| 예측 신뢰도 | % |
| 최대 변형률 | % (변형 테스트 실시간) |
| 재료 상태 | 탄성 / 소성 / 파단 |

### 하단 타임라인

- 재생 / 일시정지, 프레임 단위 탐색
- 이벤트 마커: 열 이벤트 / 응력 피크 / 변형률 수렴

## 백엔드 API

기본 주소: `http://127.0.0.1:8765`

| 엔드포인트 | 역할 |
|------------|------|
| `GET  /health` | 헬스체크 |
| `GET  /platform/status` | 외부 모델 연결 상태 확인 |
| `POST /predict` | 로컬 물성 예측 |
| `POST /platform/predict` | 사전학습 외부 모델 예측 |
| `POST /simulate` | 시뮬레이션 결과 계산 |
| `POST /import-prediction` | 외부 예측 결과 임포트 |

요청 예시:

```json
{
  "composition": { "Ni": 42, "Ti": 31, "Cr": 27 },
  "densityScale": 0.62,
  "testType": "strength",
  "scale": 1.0
}
```

백엔드는 아래 경로의 사전학습 모델을 자동으로 탐색합니다.

```
c:\ai_project\ai-materials-discovery-platform-simulation\_external\ai-materials-discovery-platform\models
```

다른 경로를 사용하려면 환경변수로 지정합니다.

```powershell
$env:AI_MATERIALS_PLATFORM_DIR = "C:\path\to\ai-materials-discovery-platform"
npm start
```
