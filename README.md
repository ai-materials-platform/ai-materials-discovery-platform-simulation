# 합금 디지털 트윈 시뮬레이션

조성 비율 기반 물성 예측과 3D 가상 시편 시뮬레이션을 통합한 Electron 데스크톱 애플리케이션입니다.  
합금 성분을 설정하면 물성값을 즉시 예측하고, ASTM 도그본 시편 또는 다양한 3D 형상에 변형·파단 시뮬레이션을 직접 수행하며 재료 시험 보고서를 PDF로 저장할 수 있습니다.

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| 조성 비율 입력 | 원소별 슬라이더로 합금 성분(%) 실시간 조정, 합계 100% 자동 제한 |
| 물성 예측 | 인장강도·항복응력·탄성계수·경도·파괴인성·밀도·용융점 등 자동 계산 |
| 3D 시편 시뮬레이션 | ASTM E8 도그본 시편에 인장·굽힘 시험 적용 |
| Von Mises 응력 시각화 | 컬러맵으로 응력 분포 시각화, 안전계수 실시간 표시 |
| 컵-앤드-콘 파단 | 오스테나이트계 스테인리스강 실제 파단과 유사한 3D 파단면 생성 |
| 응력–변형률 그래프 | 탄성→항복→변형경화→네킹 곡선 실시간 표시 |
| 재료 시험 보고서 | 8개 섹션 보고서 생성 + PDF 파일 저장 |

---

## 실행

```bash
npm install
npm start        # 프로덕션 빌드 후 Electron 앱 실행
npm run dev      # Vite 개발 서버 + Electron 동시 실행 (핫 리로드)
```

> `index.html`을 브라우저에서 직접 열면 동작하지 않습니다. Vite 번들과 Electron IPC가 필요합니다.

## 합금 설정

- **프리셋 3종**: Ni-Ti-Cr (고강도), Al-Mg-Si (경량), Fe-Cr-Ni (내열)
- 원소별 조성 슬라이더 (%, 합계 100% 초과 불가)
- 합금 목록 관리: 추가 / 삭제 / 전체 삭제
- 공정 조건: 용체화 온도 / 처리 시간 / 시험 온도
- 설정 내보내기 / 불러오기

## 물성 예측

백엔드 Python API 또는 외부 사전학습 모델로 아래 물성을 예측합니다.

| 출력값 | 단위 | 의미 |
|--------|------|------|
| 인장강도 (UTS) | MPa | 재료가 끊어지기 직전까지 버틸 수 있는 최대 응력. 높을수록 강한 재료 |
| 항복응력 (YS) | MPa | 영구적인 변형이 시작되는 응력. 이 값을 초과하면 원래 모양으로 돌아오지 않음 |
| 탄성계수 | GPa | 힘을 가했을 때 재료가 얼마나 뻣뻣한지. 높을수록 변형이 적음 |
| 연신율 | % | 파단까지 늘어나는 비율. 높을수록 잘 늘어나는 연성 재료 |
| 단면 감소율 | % | 파단 후 단면적이 줄어든 비율. 높을수록 연성이 좋고 에너지 흡수 능력이 큼 |
| 경도 (HV/HB/HRC) | — | 표면이 긁힘에 얼마나 저항하는지. 강도와 비례하는 경향 |
| 파괴인성 (KIC) | MPa·√m | 균열이 있어도 버티는 능력. 낮으면 작은 결함에도 갑자기 파단될 수 있음 |
| 밀도 | g/cm³ | 단위 부피당 질량. 낮을수록 가벼운 재료 (Al: ~2.7, Ti: ~4.5, Fe: ~7.9) |
| 용융점 | °C | 재료가 녹기 시작하는 온도. 내열 용도에서는 높을수록 유리 |
| 열전도율 | W/m·K | 열이 얼마나 잘 전달되는지. 방열 부품은 높아야, 단열 부품은 낮아야 유리 |
| 예측 신뢰도 | % | 모델이 이 조성을 얼마나 잘 학습했는지. 90% 이상이면 신뢰도 높음 |
| 격자 안정성 | % | 결정 구조가 얼마나 안정적으로 유지될지. 낮으면 상 분리나 취화 위험 |

예측 결과는 시뮬레이션의 파단 임계값·변형 스케일·응력–변형률 곡선에 직접 반영됩니다.

## 3D 시뮬레이션

### 도형

| 도형 | 설명 |
|------|------|
| 시편 (기본) | LatheGeometry 기반 ASTM E8 도그본, 컵-앤드-콘 파단 지원 |
| 구 | 등방성 합금 기본 모델 |
| 정육면체 | 블록형 구조 |
| 직육면체 | 판재·봉재 형태 |

### 물성 테스트 (시편 전용)

| 테스트 | 물리 모델 | 설명 |
|--------|-----------|------|
| 강도 (인장) | Y축 인장, Poisson 수축, 게이지 구간 집중 네킹 | 재료를 양쪽으로 당겨 끊어지는 힘을 측정 |
| 굽힘 | 3점 굽힘 단순지지 빔, X 방향 처짐 | 중앙을 눌러 휘어지는 거동 측정. 위아래 표면에서 응력 최대 |

### 시각화 모드

- **열 분포**: 주황 계열 방출광
- **응력 분포**: 빨간 계열 강조
- **X-Ray**: 반투명 이중면 렌더링
- **와이어프레임**: 메시 구조 확인

## Von Mises 응력 분석

Von Mises 응력(σ_VM)은 3D 응력 상태를 하나의 등가 응력값으로 표현하는 지표입니다.

| 항목 | 의미 |
|------|------|
| 최대 Von Mises 응력 | 시편 내부에서 가장 큰 응력이 걸리는 지점의 값 |
| 안전계수 (SF = YS / σ_VM) | 1.0 미만이면 이미 소성 변형 중 / 1.5 이상이면 안전한 설계 |
| 컬러맵 (파랑 → 초록 → 빨강) | 파란색: 응력 낮음(안전) / 빨간색: 응력 높음(위험) |

- **인장 시험**: 게이지 중앙에서 응력 집중, 넥킹 시 단면 급감으로 응력 급등
- **굽힘 시험**: 위아래 표면에서 응력 최대, 중앙 중립면에서 0 — 표면 결함이 파단을 지배

## 변형 및 파단

### 시편 변형 흐름

1. **탄성 구간** — 낮은 하중: 형상 회복 가능
2. **소성 변형** — 게이지 구간 연신, Poisson 수축으로 단면 감소
3. **네킹** — 약 55% 하중 이후 게이지 중앙부 급격 수축
4. **파단** — 임계 변위 도달 시 자동 파단

### 파단 유형

| 유형 | 특징 | 조건 |
|------|------|------|
| 연성 파단 | 끊어지기 전에 눈에 띄게 늘어남 — 사전 경고가 있어 비교적 안전 | 연신율 높음, 단면 감소율 높음 |
| 취성 파단 | 거의 변형 없이 갑자기 파단 — 충격에 취약, 위험한 파단 방식 | 연신율 낮음, 고강도 저합금 |
| 혼합형 | 연성과 취성이 혼재 | 중간 수준 |

### 컵-앤드-콘 파단 (시편 전용)

실제 오스테나이트계 스테인리스강 인장 시험과 유사한 파단면을 생성합니다.

- **상부 조각 (Cup)**: 외측 림이 아래로 휘어 컵 형태의 오목한 전단 립(shear lip) 생성
- **하부 조각 (Cone)**: 중심축이 위로 솟아 원뿔 형태의 볼록면 생성
- 양쪽 파단면에 결정립 결함과 표면 조도를 모사하는 결정론적 노이즈 적용
- 파단 후 두 조각을 각각 독립적으로 드래그 가능

## 응력–변형률 곡선

| 구간 | 모델 | 설명 |
|------|------|------|
| 탄성 | σ = E × ε (선형) | 힘을 제거하면 원래 형태로 돌아오는 구간 |
| 항복점 | 주황 점선 마커 | 여기서부터 영구 변형 시작 |
| 변형경화 | σ ∝ ε^0.52 (멱함수) | 변형될수록 점점 강해지는 구간 |
| 네킹/파단 | UTS 이후 응력 감소, 빨간 마커 | 단면이 줄어들면서 국소 변형 집중, 최종 파단 |

> **곡선 아래 면적**이 클수록 재료가 흡수할 수 있는 에너지가 많음 → 인성이 높은 재료

## 재료 시험 보고서

시뮬레이션 완료 후 **보고서 생성** 버튼을 누르면 8개 섹션의 보고서가 생성됩니다.  
**PDF 저장** 버튼으로 파일 저장 다이얼로그를 통해 PDF로 직접 저장할 수 있습니다.

| 섹션 | 내용 |
|------|------|
| 1. 합금 조성 | 원소별 조성, 역할, Schaeffler 미세조직 상 예측, 혼합 엔트로피 |
| 2. 공정 조건 | 고용화 온도·시간, 시험 온도 및 의미 |
| 3. 예측 물성값 | UTS·YS·탄성계수·경도·파괴인성·밀도 등 + 각 수치의 의미 설명 |
| 4. 응력-변형률 곡선 | 그래프 + 항복점·UTS 해설, 그래프 읽는 법 |
| 5. 3D 시뮬레이션 스냅샷 | 시편 이미지, 최대 변형률, 재료 상태 |
| 6. Von Mises 응력 | 최대 응력, 안전계수, 컬러맵 의미 설명 |
| 7. 파단 분석 | 파단 유형, 파면 형상, 파단 각도, 파괴인성 |
| 8. 예측 신뢰도 | 신뢰도, 격자 안정성, 예측 모델, 혼합 엔트로피 |

## 백엔드 API

기본 주소: `http://127.0.0.1:8765`

| 엔드포인트 | 역할 |
|------------|------|
| `GET  /health` | 헬스체크 |
| `GET  /platform/status` | 외부 모델 연결 상태 확인 |
| `POST /predict` | 로컬 물성 예측 |
| `POST /platform/predict` | 사전학습 외부 모델 예측 |
| `POST /simulate` | 시뮬레이션 결과 계산 |
| `POST /import-prediction` | 외부 예측 결과 임포트 |

요청 예시:

```json
{
  "composition": { "Ni": 42, "Ti": 31, "Cr": 27 },
  "densityScale": 0.62,
  "testType": "strength",
  "scale": 1.0
}
```

## 외부 모델 연결

백엔드는 아래 경로의 사전학습 모델을 자동으로 탐색합니다.

```
_external\ai-materials-discovery-platform\models
```

다른 경로를 사용하려면 환경변수로 지정합니다.

```powershell
$env:AI_MATERIALS_PLATFORM_DIR = "C:\path\to\ai-materials-discovery-platform"
npm start
```
